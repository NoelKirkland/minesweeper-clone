Minesweeper Instructions

- Hover to expand each subject -

Overview

In the beginning, all game tiles are concealed. Hidden among the concealed tiles, and placed at random, are a number of mines (quantity of mines and size of game grid depends on the selected difficulty). An unconcealed, non-mine tile will display a number between 1 through 8, this number represents how many mines are concealed within that tile's immediately adjacent neighboring tiles. For instance, if a revealed tile shows the number 8, that means all 8 of that tile's neighboring tiles are concealing mines. If a revealed tile shows no number at all, that means no mines are concealed within the tile's neighboring tiles.

Objective

To win the game you must reveal all non-mine tiles, leaving concealed only the tiles in which a mine is hidden. If you happen to reveal a mine then the game is lost and you must start over from a new game with mines that have been placed at new, randomly generated coordinates. As your skills improve, use the game timer and try to set a new personal record. Or, play against someone else and see who can win with the fastest time.

How to Play

  • Left click to reveal a tile
  • Right click to flag a tile you suspect is concealing a mine
  • Double click an unconcealed tile to reveal all adjacent tiles*
* the double clicked tile must have one or more flags among its adjacent tiles. The number of these flagged-tiles has to be equal to the number of mines hidden in the neighboring tiles, e.g. to double click a "2" it must be touching two flagged tiles, a "4" must be touching four flagged tiles, etc. In turn, all still-concealed, but un-flagged, neighboring tiles are revealed simultaneously

Game Specs

There are three levels of difficulty you can select from; Easy which has 8 columns of 8 tiles and 10 hidden mines, Medium which has 16 columns of 16 tiles and 40 hidden mines, and Hard which has 32 columns of 16 tiles and 99 hidden mines. Clicking the reset button (the one with the smiley face) will clear the board and assemble a new one with the difficulty specifications of whichever level you had selected last. The default level of difficulty is set to Easy.

Good Luck!
